.. _glossary:

Glossary
========

.. glossary::
   :sorted:

   Javascript
      `JavaScript <http://tools.ietf.org/html/rfc4329>`_ is an implementation
      of the `ECMAScript <http://www.ecmascript.org/>`_ language
      standard.

   Grid Layout
      One type of layout, using a grid system to lay out elements on a page.

   Layout
      The visual structure or arrangement of elements on a page.

   Layout Manager
      A central component with an API to get to all layout related
      functionality.

   Site Layout
      The general overall :term:`layout` applying to most pages of the site.

   Theme
      A visual theme, consisting of graphical appearance rules, typography and
      common graphical elements like icons.

   UI components
      Elements on a page that have some function, like presenting a text, a
      control or grouping function. Examples are a footer, a search box or a
      panel.

   Widget
     A UI control, like a date picker or a text input field.
